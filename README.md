## 🎤 Voice Cloning

NeuTTS-Studio supports voice cloning for users who want a custom voice. This lets the app copy the sound style of a sample voice and use it for new speech.

For best results:

- Use a clean voice sample
- Keep background noise low
- Use a sample with steady speaking
- Use a file in a common format like WAV or MP3

Short, clear audio samples often work best. A longer sample can help if the voice is hard to match.
